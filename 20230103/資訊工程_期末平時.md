## 觀念題與名詞解釋:精簡 摘要說明底下觀念 []
```
1. Von Neumann architecture:  范紐曼型架構，是一種將程式指令記憶體和資料記憶體合併在一起的電腦設計概念架構。
2. instruction cycle:   指令周期，是指CPU要執行一條機器指令經過的步驟，由若干機器周期組成。
3. HTML: 超文本標記語言（HyperText Markup Language)，是一種用於建立網頁的標準標記語言，用來描述網頁的內容。
4. CSS:  階層式樣式表(Cascading Style Sheets)，用來設定網頁內容的外觀，為結構化文件添加樣式的電腦語言。
5. Javascript: 用來指定網頁的行為，提供方法來操控文字、陣列、日期以及正規表示式。
6. RWD:  回應式網頁設計(Responsive Web Design)，是一種網頁設計的技術，可使網站在不同的裝置上瀏覽時對應不同解析度皆有適合的呈現。
```
